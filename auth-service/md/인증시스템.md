# 인증 시스템

---

- 세션과 쿠키를 사용한 전통적인 인증 시스템은 모바일 애플리케이션에서 유효하게 사용할 수 없음(공유 불가)
- 렌더링된 HTML 페이지가 반환되지만, 모바일 애플리케이션에서는 JSON(or XML)과 같은 포멧 필요


---

- Java에서 발급되어 있는 인증방법을 다른 언어에서 쓰기 위한 방법 -> Token
- 세션 정보 같은 것들을 다른 포맷, 공통적인 JSON, XML 같은 포맷으로 변환 필요

---

### Token 기반 인증 시스템
- 서버에서 인증 절차를 거친 다음 반환을 시켜주고자 할 때 이전처럼 session을 발급해주는 것이 아닌 Token을 발급
- 클라이언트에서는 웹브라우저, 모바일 디바이스 상관 없이 토큰을 받은 후 추가적인 서비스 요청 가능

### JWT(JSON Web Token)
- 인증 헤더 내에서 사용되는 토큰 포맷
- 두 개의 시스템끼리 안전한 방법으로 통신 가능
